Abstract

Video world models have shown immense potential in simulating the physical world, yet existing memory mechanisms primarily treat environments as static canvases. When dynamic subjects hide out of sight and later re-emerge, current methods often struggle, leading to frozen, distorted, or vanishing subjects. We introduce Hybrid Memory, a paradigm requiring models to act as precise archivists for static backgrounds and vigilant trackers for dynamic subjects, ensuring motion continuity during out-of-view intervals. To facilitate research, we construct HM-World, a large-scale dataset with 59K high-fidelity clips, decoupled camera/subject trajectories, 17 scenes, 49 subjects, and designed exit-entry events to evaluate hybrid coherence. We further propose HyDRA, a memory architecture that compresses context into memory tokens and retrieves relevant cues via spatiotemporal relevance-driven attention. Experiments on HM-World show strong improvements in dynamic subject consistency and overall generation quality.

Introduction

Memory mechanisms are a critical frontier for video world models, because memory capacity directly impacts spatial and temporal consistency under viewpoint shifts and long-horizon generation. Recent work improves memory via retrieval and compression, but often treats the environment as a static canvas. In the real world, scenes are populated by dynamic subjects governed by independent motion logic.

When dynamic subjects exit the camera’s field of view and later re-enter, static-centric memory methods frequently fail: returning subjects become frozen, distorted, or vanish. We propose Hybrid Memory, which requires the model to anchor static backgrounds while continuously tracking and extrapolating dynamic subjects during out-of-view intervals, so that identity and motion remain coherent upon re-entry.

HM-World Dataset

A benchmark dedicated to hybrid memory with exit-entry events.

Natural videos with clean, frequent exit-and-re-entry events are scarce, making systematic evaluation of hybrid memory difficult. We introduce HM-World, a large-scale dataset purpose-built to train and evaluate hybrid memory for video world models.

HM-World contains 59,225 high-fidelity clips rendered in Unreal Engine 5 with decoupled camera trajectories and subject motions, creating many natural out-of-view intervals. The dataset covers 17 diverse scenes and 49 distinct subjects (humans and animals), designed subject trajectories, and 28 camera motion patterns that induce exit-entry events. Each clip is annotated with camera poses, per-frame subject positions, and timestamps of exit/entry events.

HyDRA Method

Hybrid Dynamic Retrieval Attention for consistent re-entry.

We propose HyDRA (Hybrid Dynamic Retrieval Attention), a memory mechanism designed to preserve dynamic subject identity and motion while maintaining static background coherence under camera motion.

HyDRA first compresses context latents into compact, motion-aware memory tokens using a 3D-convolution-based memory tokenizer. During generation, a spatiotemporal relevance-driven retrieval module computes affinity between target queries and memory keys, selects top-k relevant tokens, and fuses them with a local temporal window to stabilize denoising. This selective retrieval helps the model “rediscover” hidden subjects and continue their trajectories after out-of-view intervals.
